Welcome to the Library Management System project! This application helps manage library operations efficiently, allowing users to add books, update ratings, and view book lists.

About the Project

The Library Management System is a Java-based application that allows users to manage library operations easily. Key features include:

  • Add new books
  • Update book ratings
  • View all books in the library
  • Display the top-rated book

This application utilizes Java's Collections Framework for effective data management.
